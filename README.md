
# MOUNTAINRENT

Sistem Informasi Penyewaan Peralatan Pendakian Berbasis Web.

## Teknologi

* HTML
* CSS
* JavaScript
* PHP
* MySQL
* XAMPP
* InfinityFree

## Fitur

* Input data penyewa
* Penyewaan alat pendakian
* Perhitungan harga otomatis
* Riwayat penyewaan
* History pengembalian
* Detail barang yang disewa
